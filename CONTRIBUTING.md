# Contribuindo

Obrigado por contribuir com o projeto Revitalize!

Diretrizes rápidas:

- Fork e clone o repositório.
- Crie uma branch por feature/bugfix: feature/nome-descritivo
- Faça commits pequenos e claros.
- Abra Pull Request apontando para main e descreva o objetivo.
- Execute lint/build antes de submeter.

Se quiser ajuda para configurar um ambiente local no seu computador, descreva seu SO e eu forneço passos detalhados.
